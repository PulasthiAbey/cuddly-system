Phase 1: Database Setup (Supabase)

Before building the UI, we need a place to store your data. Supabase is the best choice since it uses PostgreSQL and integrates perfectly with Next.js.

  1. Go to Supabase and create a free project.
  2. Run this SQL command in their SQL Editor to create your first table for tracking morning weight and sleep:
CREATE TABLE daily_metrics (
    id UUID DEFAULT uuid_generate_v4() PRIMARY KEY,
    date DATE UNIQUE NOT NULL,
    weight_kg DECIMAL(5,2),
    sleep_hours DECIMAL(4,2),
    fasting_hours DECIMAL(4,2),
    created_at TIMESTAMP WITH TIME ZONE DEFAULT NOW()
);

Next, install the Supabase client in your repo: npm install @supabase/supabase-js

Phase 2: Build the Main Dashboard UI

We will create a clean dashboard in src/app/page.tsx to log your daily metrics easily.

  • Weight Input Component: A simple number input that saves your morning weight (target: break 99kg!).
  • Sleep Tracker Component: A slider or input to log hours slept (target: 7-8 hours).
  • Fasting Timer (Optional): A simple state-driven timer to track your 7:00 PM to 11:00 AM fasting window.
